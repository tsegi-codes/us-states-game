# 🗺️ U.S. States Game

A geography quiz game built with Python where you guess all 50 U.S. states 
and see them placed on a map in real time.

## How to Play
- Type a state name to place it on the map
- You get 5 incorrect guesses before game over
- Type "Exit" to quit early
- Any unguessed states are saved to `states_to_learn.csv` so you can study them

## What I Learned
- Reading CSV data using Pandas
- Combining a dataset with a Turtle graphics GUI
- Writing filtered results back to a new CSV file

## How to run the project
1. Install Python
2. Install dependencies: `pip install pandas`
3. Clone the repository
4. Run:
   ```bash
   python main.py
   ```
